It also provides integration with other Spring ecosystem project such as Spring Integration and Spring Batch enabling you to develop solutions for big data ingest/export and Hadoop workflow orchestration. Check out the new book from O'Reilly Media [Spring Data: Modern Data Access for Enterprise Java](http://shop.oreilly.com/product/0636920024767.do#) that contains several chapters on using Spring for Apache Hadoop. Sample code for the book is also available on github [here](https://github.com/SpringSource/spring-data-book). ## Features * Support to create Hadoop applications that are configured using Dependency Injection and run as standard Java applications vs. using Hadoop command line utilities. * Create and configure applications that use Java MapReduce, Streaming, Hive, Pig, Cascading, or HBase * Extensions to Spring Batch to support creating Hadoop based workflows for any type of Hadoop Job or HDFS operation. * Script HDFS operations using any JVM based scripting language * DAO support (Template & Callbacks) for HBase * Cascading Taps for Spring & Spring Integration * Support for Hadoop Security ## Quick Start {% include download_widget.md %} {% endcapture %} {% capture related_resources %} ### Sample Projects * [Spring for Hadoop Samples](https://github.com/SpringSource/spring-hadoop-samples) {% endcapture %} {% include project_page.html %}
